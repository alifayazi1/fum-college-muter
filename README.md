*Would you care to sign up for a fantastic beginner's Python course? How about advanced Python? No? What about an English language test? No to English? Then how about Arabic? Not a fan of languages? Laptop repair, then—surely that comes in handy! Not that either? Producing AI slop is all the rage now, we've got a course for that. None of those sound good? Well then, how about you sign up for our special course in MDF carpentry or smart automation and water management on the farm!*

If you are also tired of seeing countless "College" spam on your portal with subjects like these, this extension is for you.

#### What does this extension do?
This tool is designed to remove College spam messages from the **Ferdowsi University of Mashhad (FUM) Pooya Portal**. Here is how it works:
1. It scans all your notifications in the "News" section of the portal for the word **"کالج"** (College).
2. If such messages are found, a window pops up and lists them for you.
3. You can select which messages to delete (by default, all are selected so you can clear them all with just one click).
4. After confirmation, the extension automatically clicks the trash icon for each message and bypasses the browser's confirmation popups to speed up the process.
5. **Note:** This extension works entirely locally on your device and does not share any data with any services, individuals, or other websites.

#### How to install on Google Chrome and Microsoft Edge:
1. Download the `fum-college-muter.zip` file from the Releases section and extract it into a folder.
2. Open Chrome or Edge and go to `chrome://extensions/` (Chrome) or `edge://extensions/` (Edge).
3. Enable **Developer mode** in the top right corner.
4. Click the **Load unpacked** button.
5. Select the folder created in step 1.
6. The extension is now active. Open a new tab and log into the Pooya portal. If any College messages exist, a window will appear allowing you to delete all promotional messages at once.

#### How to install on Firefox (Desktop):
1. Download the `fum-college-muter.xpi` file from the Releases section.
2. In Firefox, navigate to `about:addons`.
3. Click the gear icon (Settings) and select **Install Add-on From File...**.
4. Select the `.xpi` file and confirm. (Note: If the standard version of Firefox blocks permanent installation of unsigned files, use the `about:debugging` method to load it temporarily).

#### How to install on Firefox Android:
1. In the Firefox mobile browser, go to **Settings** and then **About Firefox**.
2. **Tap the Firefox logo 5 times** to enable the hidden **Custom Add-on Collection** menu.
3. Go back to the Add-ons section in Settings. You can now add the extension using its `.xpi` file.
4. **Easier Method:** Use browsers like **Mull** or **Iceraven**, which allow direct installation of `.xpi` files through the Add-ons menu.
